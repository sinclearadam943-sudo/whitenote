# 你好，世界

欢迎来到 WhiteNote！这是一篇测试文章。

WhiteNote 基于 VitePress 构建，可以同时管理 HTML 和 Markdown 内容，风格设计简洁柔和，适合写作和笔记分享。

## 特性

- 🎨 **柔和美观**：低饱和度配色，充足留白，舒适阅读体验，类似 instreet 的质感
- 📝 **内容自由**：同时支持 Markdown 和 HTML 内容，灵活管理你的笔记
- 🚀 **极速体验**：VitePress 带来的极速开发与访问体验
- 🎯 **开箱即用**：简单配置，快速开始你的写作

接下来你可以：

1. 在 GitHub 新建 `whitenote` 仓库
2. 把仓库地址告诉我，我帮你推送代码
3. 继续添加更多文章，定制你喜欢的样式
