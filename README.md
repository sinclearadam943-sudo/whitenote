# WhiteNote 📝

一个基于 **VitePress** 构建的干净柔和的笔记/博客站点，支持同时管理 Markdown 和 HTML 内容，风格设计偏向柔和低饱和，类似 instreet 质感。

## 特性

- ✨ 柔和低饱和配色，舒适阅读体验
- 📝 同时支持 Markdown 和 HTML 内容管理
- 🚀 基于 VitePress，极速开发与构建
- 🎨 默认内置柔和主题风格，可灵活定制
- 📱 移动端友好自适应

## 项目结构

```
.
├── docs/
│   ├── .vitepress/
│   │   ├── config.ts       # VitePress 配置
│   │   ├── theme/          # 主题定制
│   │   └── public/         # 静态资源
│   ├── index.md            # 首页
│   └── posts/              # 文章目录
├── .gitignore
└── package.json
```

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发，启动预览
npm run dev

# 构建静态网站
npm run build

# 预览构建结果
npm run preview
```

## 风格特点

- 主色调：柔和绿色 `#6b9080`，低饱和度
- 整体背景浅灰，减少视觉疲劳
- 保持 VitePress 原生干净体验，定制柔和质感
- 适配写作、笔记、博客多种场景
